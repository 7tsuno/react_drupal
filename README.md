# react_drupal

## 事前条件

* npm, nodeがインストールされていること。
* gitがインストールされていること。

## 起動

このプロジェクトをgitでcloneします。

```
git clone https://github.com/7tsuno/react_drupal
```

プロジェクトのルートディレクトリに移動します。

```
cd react_drupal
```

npm(ノードパッケージマネージャ)を利用し、依存するライブラリをインストールします。

```
npm install
```

起動します。

```
npm start
```

ブラウザが立ち上がったら成功です。
